# XDSEC 招新系统

XDSEC 招新系统的前端项目，面向候选人、面试官与管理员提供统一的入口与操作体验。

## 项目功能
### 账号与身份
- 支持注册、登录、退出登录
- 支持重置/修改密码，保障账号安全
- 个人资料可维护昵称、签名与方向等信息

### 公告系统
- 面试官可发布公告并配置可见范围
- 公告列表清晰展示发布者、发布时间与内容
- 支持公告内容置顶、编辑与更新

### 面试申请
- 候选人可提交面试申请，填写基础信息与简历
- 支持多方向选择，并提供简历预览
- 申请流程清晰，便于候选人随时补充或更新信息

### 面试官工作区
- 候选人列表与详情查看，支持设置通过方向与面试状态
- 任务管理独立页面：布置任务、编辑任务与查看任务反馈
- 公告管理独立页面：发布公告、调整可见范围与更新内容
- 候选人详情页聚合简历、任务、评论等信息，便于集中评估

### 任务系统
- 面试官创建任务，面试者提交任务报告
- 任务详情统一展示标题、描述、更新时间与报告内容
- 任务列表更易筛选与定位目标候选人

### 主题与体验
- 支持 Markdown 渲染与代码高亮，提升可读性
- 支持浅色/深色模式切换
- 页面排版统一，信息密度适中，便于集中阅读
